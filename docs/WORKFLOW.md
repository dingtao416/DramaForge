### 6 阶段线性流水线

```
输入: 用户创建项目 (题材/主题/时长/集数/风格)
                │
    ┌───────────▼───────────┐
    │ Stage 1: 生成大纲      │ ScriptAgent → ai_hub.chat.complete_json()
    │ • 标题/概要/角色/分集   │ 输出: JSON {title, characters, episodes}
    └───────────┬───────────┘
                │ 🔍 人工审核点
    ┌───────────▼───────────┐
    │ Stage 2: 生成剧本      │ ScriptAgent → ai_hub.chat.complete()
    │ • 场景/对白/旁白/动作   │ 输出: Markdown 格式剧本文本
    └───────────┬───────────┘
                │ 🔍 人工审核点
    ┌───────────▼───────────┐
    │ Stage 3: 分镜拆解      │ StoryboardAgent → ai_hub.chat.complete_json()
    │ • 画面描述/旁白/时长    │ 输出: JSON [{sequence, image_prompt, narration, ...}]
    │ • 英文 image_prompt    │ 8~20 个分镜面板
    └───────────┬───────────┘
                │ 🔍 人工审核点 (逐条审核/编辑分镜)
    ┌───────────▼───────────┐
    │ Stage 4a: 图片生成     │ ImageAgent → ai_hub.image.generate()
    │ • sora-image $0.01/张  │ 输出: PNG 文件 (1024×1792)
    ├───────────────────────┤
    │ Stage 4b: 语音合成     │ TTSAgent → ai_hub.tts.speak()
    │ • tts-1-hd, nova 女声  │ 输出: MP3 文件 + 时长
    └───────────┬───────────┘
                │
    ┌───────────▼───────────┐
    │ Stage 5: 视频合成      │ VideoAgent → moviepy 本地合成
    │ • 图片+音频+字幕→MP4   │ 输出: 1080×1920 竖屏视频
    │ • ffmpeg 编码          │ 可选: Sora 2 / Veo 3.1 AI视频片段
    └───────────┬───────────┘
                │
                ▼
输出: 完整短剧视频 (.mp4)
```

### 当前已验证的 AI 服务

| 服务 | 状态 | 模型 | 耗时 | 费用 |
|------|------|------|------|------|
| LLM 文本生成 | ✅ 通过 | gpt-4o | 3.6s | ~$0.01 |
| 图片生成 | ✅ 通过 | sora-image | ~90s | $0.01/张 |
| TTS 语音合成 | ✅ 通过 | tts-1-hd | 4.5s | ~$0.01 |
| 视频生成 | ⚠️ 待验证 | veo-3.1-fast | - | 按次 |
| 视频合成 | ✅ 已实现 | moviepy 本地 | - | 免费 |

### 当前痛点

1. **线性串行** — 6 阶段串行执行，生成 20 集短剧需数小时
2. **静态画面** — Stage 5 只是图片+音频拼接，没有动态效果
3. **角色一致性差** — 不同分镜中同一角色外貌可能不一致
4. **无背景音乐** — 只有旁白/对白，缺少 BGM 和音效
5. **缺少字幕样式** — 字幕仅 TextClip，无样式/动画
6. **单语言** — 仅中文剧本，未支持多语种配音
