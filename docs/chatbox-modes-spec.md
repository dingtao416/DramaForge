## 一、模式选择器（Mode Selector）

对话框工具栏中的"模式"按钮，点击后弹出模式选择浮层，共 **6 种模式**。
切换模式后，工具栏中的**图标组合、提示文字、附加功能按钮**都会动态变化。

### 模式列表

| # | 模式名称 | 标签 | 描述 |
|---|---------|------|------|
| 1 | **Agent 模式** | — | 全能创作 Agent，图片、短片、长视频一站式创作 |
| 2 | **短剧 Agent** | `New` | 一键进入短剧创作工作台，快速开始分镜与剧情生成 |
| 3 | **沉浸式短片** | — | 15秒内音画同出短视频，一句话秒出片 |
| 4 | **智能长视频 2.0** | — | 自动多分镜编排，轻松生成高质量长片 |
| 5 | **生成图片** | — | 输入描述即刻出图，快速验证创意灵感 |
| 6 | **智能长视频** | — | 基础长视频流程，速度稳定均衡 |

## 四、弹出菜单规格

### 4.1 上传菜单（+ 按钮）

**触发：** 点击 `上传参考素材` 或 `+` 按钮

| 选项 | 图标 | 说明 |
|------|------|------|
| 本地上传 | 📤 上传箭头 | 打开文件选择器，上传本地图片/视频 |
| 从资产库选择 | 📁 文件夹 | 跳转到资产库，选择已有素材 |

---

### 4.2 模型偏好面板（模型图标 + ▾）

**触发：** 点击模型偏好按钮（Agent 模式下可见）

**面板结构：**

```
┌─────────────────────────────────────┐
│ 模型偏好                    自动 [🔘] │  ← 头部：标题 + 自动开关 Toggle
├─────────────────────────────────────┤
│ [视频]  [图片]                       │  ← Tab 切换
├─────────────────────────────────────┤
│ 视频模型                             │  ← 分区标签
│ ≫ Seedance 2.0 Fast                │  ← 模型项（图标 + 名称 + 描述）
│   更快更实惠的视频生成模型。            │
├─────────────────────────────────────┤
│ 图片模型（切换 Tab 后显示）            │
│ ≫ Seedream 5.0 Lite                │
│   字节跳动最新图像生成模型，画质更强。   │
│ ≫ Seedream 4.5                     │
│   字节跳动图像生成模型。               │
│ ≫ Seedream 4.1                     │
│   字节跳动图像生成模型。               │
└─────────────────────────────────────┘
```

---

### 4.3 画幅比例选择（⊞ 比例按钮）

**触发：** 点击 `⊞ 比例` 按钮（**所有模式通用**）

| 选项 | 图标 | 说明 |
|------|------|------|
| ⊞ 自动 | 田字格 | 默认，自动判断最佳比例 ✓ |
| ▭ 16:9 (横屏) | 横向矩形 | 标准横屏比例 |
| ▯ 9:16 (竖屏) | 纵向矩形 | 竖屏/短视频比例 |
| ▭ 4:3 | 横向矩形 | 传统横屏比例 |
| ▯ 3:4 | 纵向矩形 | 竖屏比例 |

**交互：** 单选，选中项显示紫色勾 ✓

---

### 4.4 预设提示词面板（📄 按钮）

**触发：** 点击预设提示词按钮

**面板结构：**

```
┌──────────────────────────────────────┐
│ ⊞ 预设提示词                   📥 📤 │  ← 头部：标题 + 导入/导出图标
├──────────────────────────────────────┤
│ [🔍 搜索预设...]          [+ 新建]   │  ← 搜索框 + 新建按钮
├──────────────────────────────────────┤
│            📖                        │
│   暂无预设，点击新建创建第一个预设      │  ← 空状态
└──────────────────────────────────────┘
```

**功能：**
- **搜索框：** 支持关键词搜索已有预设
- **+ 新建：** 创建新的提示词预设模板
- **📥 导入 / 📤 导出：** 预设数据导入导出
- **预设列表：** 展示已保存的提示词模板，点击可快速填入输入框

## 五、通用 UI 规格

### 工具栏按钮尺寸

| 类型 | 高度 | 宽度 | 圆角 | 边框 |
|------|------|------|------|------|
| 模式切换按钮（带文字） | 42px | 自适应 | 全圆角 (rounded-full) | 1px solid #E5E5E5 |
| 功能按钮（带文字） | 42px | 自适应 | 全圆角 | 1px solid #E5E5E5 |
| 圆形图标按钮 | 42px | 42px | 全圆角 | 1px solid #E5E5E5 |

### 弹出面板尺寸

| 面板 | 宽度 | 圆角 | 阴影 |
|------|------|------|------|
| 上传菜单 | 200px | 14px | 0 8px 32px rgba(0,0,0,0.1) |
| 模型偏好 | 320px | 14px | 同上 |
| 画幅比例 | 200px | 14px | 同上 |
| 预设提示词 | 360px | 14px | 同上 |

### 弹出方向

所有弹出面板从按钮**上方**弹出（`bottom: calc(100% + 8px)`），带 0.15s 淡入动画。

## 六、后续 UI 设计建议

1. **优先实现：** Agent 模式 + 沉浸式短片模式的完整工具栏切换
2. **需要设计的交互：**
   - 模式选择浮层的选中态（紫色勾 ✓）
   - 沉浸式短片模式下的 `@引用素材`、`参考`、`时长选择` 弹出面板
   - 模型版本标签的样式（灰底圆角 tag）
   - 上方提示条的样式（模型信息 + 计费说明）
3. **`⊞ 比例` 按钮为全局通用组件，可封装为独立组件复用**
