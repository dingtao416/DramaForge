## 1. 平台概览

小云雀是剪映推出的 **AI 视频创作 Agent 平台**，定位为"一站式 AI 成片工具"。其核心设计理念是：

- **对话驱动**：用户通过自然语言描述需求，Agent 自动完成全流程生产
- **一键成片**：从文字/创意到完整视频的端到端自动化
- **多模态 AI**：整合了文本生成、图片生成、视频生成、语音合成等全部 AI 能力
- **角色一致性**：通过资产库和角色绑定，确保同一角色在全剧集中的视觉一致性

## 2. 核心架构：对话式 Agent 交互

小云雀的核心交互模式是 **Chat-based Agent**，而非传统的表单/向导式。

### 2.1 输入区工具栏

| 图标/按钮 | 功能 | 说明 |
|----------|------|------|
| `+` | 上传参考素材 | 支持上传图片/视频作为创作参考 |
| `Agent 模式 ▼` | 创作模式切换 | Agent 模式 = AI 自主规划全流程 |
| `模型选择 ▼` | 视频模型选择 | 可切换 Seedance 2.0 Fast / 标准 等 |
| `网格图标` | 画面网格/构图 | 辅助画面构图 |
| `显示器图标` | 画面比例 | 16:9 / 9:16 / 1:1 等 |
| `✦ 一键优化提示词` | 提示词优化 | AI 自动润色和增强用户输入的提示词 |
| `预设提示词` | 保存/复用提示词 | 支持新建、搜索、导入导出预设 |

### 2.2 创作模式

| 模式 | 交互方式 | 产出 |
|------|---------|------|
| **Agent 模式** | 输入创意 → AI 自主完成全部步骤 | 完整视频 |
| **沉浸式短片** | 详细描述单个镜头 → 生成视频 | 单段视频 |
| **爆款复刻** | 粘贴参考视频链接 → AI 解析 → 复刻 | 风格化视频 |

### 2.3 快捷提示词标签

平台底部提供热门创作灵感标签，一键填充提示词：
- `漫剧：超绝人物特写`
- `香港电影·996`
- `猫狗偷玩手机一秒装睡`
- `潮汕功夫茶宣传片`
- `爆款手办生成`
- `柴犬历险记`

## 3. 四大核心功能

### 3.1 短剧 Agent

> **入口**: 常用功能 → "短剧 Agent" 卡片
> **URL**: `xyq.jianying.com/novel/list`
> **定位**: 全面应用 Seedance 2.0 模型，一键直出整部剧！

这是平台的 **旗舰功能**，也是与 DramaForge 最直接对标的模块。

#### 3.1.1 输入方式 (二选一)

| 方式 | 说明 | 限制 |
|------|------|------|
| **上传我的剧本** | 上传 `.docx` 格式的剧本文件，拖拽上传 | 不超过 10 万字 |
| **AI 生成剧本** | 文本框输入故事构想，AI 自动生成完整剧本 | 限免 1 次 |

**AI 生成剧本提示词要素**：
- 故事设定
- 主角特征
- 剧情脉络
- 最终结局

#### 3.1.2 三步工作流 ⭐

短剧 Agent 采用 **三步流水线** 工作流，每一步之间有明确的审核/调整节点：

```
Step 1             Step 2               Step 3
┌──────────┐      ┌──────────────┐     ┌──────────┐
│ 剧本大纲  │ ──→  │ 角色和场景    │ ──→ │ 分集视频  │
│          │      │              │     │          │
│ ✓ 视频风格│      │ 全部角色 (N)  │     │ 共 N 集   │
│ ✓ 画面比例│      │ 全部场景 (N)  │     │ 预览/编辑 │
│ ✓ 剧本摘要│      │ 可编辑/替换   │     │ 导出/合成 │
│ ✓ 分集剧本│      │              │     │          │
└──────────┘      └──────────────┘     └──────────┘
     ↓                   ↓                   ↓
   可编辑             可调整角色           可重新生成
  改写旁白型          可替换场景图         可编辑脚本
```

#### Step 1: 剧本大纲

从剧本自动提取/生成以下结构化信息：

| 字段 | 示例 |
|------|------|
| **视频风格** | 真人写实, 电视风格, 暖色调 |
| **画面比例** | 16:9 |
| **主角** | 沈念安 |
| **故事类型** | 女频 |
| **故事梗概** | [地位反差/身份错位 豪门养女...] |
| **故事背景** | 现代都市。主要场景包括... |
| **故事设定** | 主角被神秘人所救，条件是必须隐姓埋名... |
| **一句话故事** | 在自己盛大的订婚宴上... |
| **剧本内容** | 完整的分集对话剧本（含舞台指示△） |

**关键功能**：
- `改写为旁白型剧本`：将对话型剧本一键转换为旁白叙述型
- 底部提示："剧本内容整理完毕，可以进行下一步了"

**剧本格式规范**（从示例中提取的格式）：
```
第N集
N-序号 时间(日/夜) 场景(内/外) 场景名称
人物：角色A, 角色B, ...
△ (舞台指示/动作描写)
角色名（情绪/动作）：台词内容
角色名OS：内心独白
角色名VO：画外音
【闪回】 ... 【闪出】
```

#### Step 2: 角色和场景

**角色管理**：
- 自动从剧本中识别所有角色（示例中提取了 11 个角色）
- 每个角色自动生成 **全身形象参考图**（AI 生图）
- 标注主角/配角
- 支持 **编辑/替换** 角色形象
- 每个角色可有多个形象（换装）
- 角色形象会应用到整部剧集中，确保 **一致性**

**场景管理**：
- 自动从剧本中识别所有场景
- 每个场景生成场景参考图（可多张）
- 示例：沈念安的豪华公寓(3图)、设计工作室门口(1图)、豪华酒店宴会厅(2图)
- 场景图同样在全剧集中复用

**底部提示**: "角色和场景设定会应用到整部剧集中，建议调整完毕后再继续哦"

#### Step 3: 分集视频

**视频列表视图**：

| 字段 | 说明 |
|------|------|
| 集数 | 第 N 集 |
| 标题 | 自动生成的集名（如"订婚惊变妹妹夺爱"） |
| 角色数 | 该集涉及的角色数量 |
| 场景数 | 该集涉及的场景数量 |
| 分镜数 | 自动拆解的分镜数量 |
| 时长 | 如 01:25 |
| 操作 | **预览** / **编辑** / **导出** |
| 批量操作 | 多选按钮 |

**积分消耗**: 分镜脚本生成 = 100 字符消耗 2 积分

**核心元素解析**：

##### a) 资产库面板（左侧）
- **角色列表**：显示该集所有角色的参考图，可编辑
- **旁白设置**：旁白形象配置
- **场景列表**：该集用到的场景参考图，可编辑

##### b) 分镜脚本（中间）
每个片段(Segment)包含多个分镜(Shot)，每个分镜的详细描述：

| 分镜属性 | 说明 | 示例 |
|---------|------|------|
| **时长** | 可调节 0-15s | ⊙ 6.0s |
| **时间** | 日/夜 | 日 |
| **场景图片** | @引用场景资产 | @豪华酒店宴会厅_0 |
| **镜头类型** | 景别+角度 | 中景镜头，从舞台正下方略仰视角度 |
| **角色** | @引用角色资产 | @司仪-基础形象-基础形象 |
| **角色动作** | 姿势/表情/视线 | 手持话筒，面带职业微笑 |
| **台词** | 角色对话 | 「今天，是沈家千金...」 |
| **音色** | TTS 风格描述 | 男声，青年音色，音调中等，音色明亮圆润 |
| **背景** | 画面背景描述 | 鲜花簇拥的舞台和璀璨的水晶灯 |
| **镜头动作** | 运镜方式 | 镜头静止 / 缓慢推进 |
| **分镜过渡** | 镜头切换方式 | 镜头平滑切换，焦点始终在舞台中央 |

**@ 引用系统**：
- 输入 `@` 可快速引用角色、场景、素材
- 引用的资产会自动绑定到生成的视频中，确保视觉一致性

##### c) 视频预览（右侧）
- 生成的视频段落实时预览
- 支持播放/暂停
- 显示时长

##### d) 时间线（底部）
- 所有片段以时间线形式排列
- 每个片段显示缩略图和时长
- 支持多选操作
- 显示总时长

**操作按钮**：

| 按钮 | 功能 |
|------|------|
| `Seedance 2.0 • Fast ▼` | 选择生成模型/速度 |
| `导出` | 导出单集视频 |

### 3.2 Seedance 2.0 视频生成

> **入口**: 常用功能 → "Seedance 2.0" 卡片
> **定位**: 首发试用字节跳动自研视频生成模型

Seedance 2.0 是小云雀的底层视频生成引擎。

**模型版本**：
| 版本 | 特点 |
|------|------|
| Seedance 2.0 • Fast | 快速生成，质量稍低 |
| Seedance 2.0 • Standard | 标准速度，更高质量 |

**使用方式**：
- 直接在主对话框中描述画面内容
- 选择"沉浸式短片"模式
- 支持 @引用素材
- 支持设置参考视频/图片
- 支持设置时长（如 10s）
- 按 1 秒 5 积分扣除（沉浸式短片模式）

### 3.3 爆款复刻

> **入口**: 常用功能 → "爆款复刻" 卡片
> **URL 参数**: `mode=create&feature=imitation`
> **定位**: 自动解析爆点，参考文案/主题/画风

**工作流程**：
```
粘贴参考视频链接 → AI 解析视频爆点 → 生成类似风格的新视频
     或
从本地上传视频文件 → 同上
```

**输入方式**：
- 粘贴视频链接（支持 **抖音、头条、西瓜** 三个平台）
- 从本地上传视频文件

**解析维度**：
- 爆点原因分析
- 文案风格提取
- 画面主题识别
- 画风/色调参考

**输入框预填提示词**: "解析参考视频的爆点原因并借鉴在视频中"

> **入口**: 常用功能 → "一镜到底" 卡片

## 4. 资产库系统

> **URL**: `xyq.jianying.com/asset`

资产库是小云雀的 **全局素材管理中心**，所有创作过程中产生/上传的素材都集中管理。

### 4.1 资产分类

| 分类 | 说明 |
|------|------|
| **资产** | 所有类型的素材（图片、视频等） |
| **人物角色** | 角色形象资产，用于跨项目复用 |

### 4.2 资产属性

| 属性 | 说明 |
|------|------|
| 文件名 | 如 `female_23.png` |
| 来源 | `创作生成` / `用户上传` |
| 时间 | 创建/上传时间 |
| 标签 | 支持添加和编辑标签，按标签筛选 |

### 4.3 资产操作

- **新增**: 上传新素材
- **刷新**: 刷新列表
- **视图切换**: Grid / List 布局
- **筛选**: 按标签筛选
- **排序**: 按时间倒序展示
- **删除**: 支持删除资产（新功能 2026-03-30）

### 4.4 跨项目引用

资产库中的角色/场景可以在不同项目中复用：
- 在分镜编辑器中通过 `@` 引用
- 确保角色在多个视频中保持一致外观

以示例项目为例（5集短剧，总时长约 6:26）：
- 总视频时长：386 秒
- 视频生成积分：386 × 11 = **4,246 积分**
- 月费会员积分：1,200 + 每日赠送
- 预计需要额外充值积分

## 6. 对 DramaForge 重构的启示

### 6.1 核心设计理念对齐

| 维度 | 小云雀做法 | DramaForge 当前 | 建议改进 |
|------|----------|----------------|---------|
| **交互方式** | 对话式 Agent | API/脚本调用 | 🔴 增加对话式前端 |
| **工作流** | 三步流水线 | 六步线性 | 🟡 精简为三步+可选步骤 |
| **角色一致性** | @引用+资产库 | 无 | 🔴 构建角色资产系统 |
| **场景管理** | 预生成+可编辑 | 无 | 🔴 增加场景库 |
| **分镜粒度** | 精确到秒级+镜头语言 | 粗粒度 | 🟡 增强分镜描述 |
| **视频模型** | Seedance 2.0 (自研) | 多模型 fallback | 🟢 保持灵活性 |
| **实时预览** | 有 | 无 | 🟡 增加预览机制 |
| **批量操作** | 多选/批量导出 | 无 | 🟡 增加批量处理 |

### 6.2 需要重点借鉴的功能

#### 🔴 P0 - 必须实现

1. **三步核心工作流**
   ```
   剧本 → 角色&场景 → 分集视频
   ```
   - 每步之间有人工审核/调整节点
   - 前一步的设置影响后续所有步骤

2. **角色资产系统**
   - 基于剧本自动提取角色列表
   - 为每个角色生成参考形象图
   - 角色形象在全剧集中复用（一致性）
   - 支持编辑/替换角色形象

3. **分镜脚本的结构化描述**
   - 每个分镜包含：时长、场景、镜头类型、角色、动作、台词、音色、运镜
   - 使用 @引用 绑定角色和场景资产
   - 支持编辑和重新生成

4. **剧本格式标准化**
   ```
   第N集
   场景号 时间 内/外 场景名
   人物：角色列表
   △ 动作指示
   角色名（情绪）：台词
   角色名OS：内心独白
   ```

#### 🟡 P1 - 重要但可迭代

5. **场景库管理**
   - 从剧本自动识别场景
   - 为每个场景生成参考图
   - 同一场景在不同集中保持视觉一致

6. **视频预览和时间线**
   - 生成后即时预览
   - 时间线展示所有片段
   - 支持单片段重新生成

7. **改写为旁白型剧本**
   - 对话型 → 旁白型的一键转换
   - 适配不同的视频风格（推文/短剧/纪录片）

8. **提示词优化系统**
   - 一键优化提示词
   - 预设提示词库（可保存/复用/导入导出）

#### 🟢 P2 - 锦上添花

9. **爆款复刻**（参考视频解析 → 风格迁移）
10. **一镜到底**（多图转场）
11. **合成配置**（字幕、BGM、片头片尾等后期参数）
12. **积分体系**（用量计量/限制）

### 6.4 数据模型建议

```python
# 基于小云雀的产品设计，建议的核心数据模型

class Project:
    title: str              # "从弃女到巅峰：苏家千金归来"
    style: VideoStyle       # 真人写实/动漫/3D
    aspect_ratio: str       # "16:9"
    status: StepEnum        # script / assets / video

class Script:
    project_id: str
    protagonist: str        # 主角名
    genre: str              # 女频/男频/科幻/...
    synopsis: str           # 故事梗概
    background: str         # 故事背景
    setting: str            # 故事设定
    one_liner: str          # 一句话故事
    episodes: List[Episode]
    script_type: str        # "dialogue" / "narration"

class Episode:
    number: int
    title: str              # "订婚惊变妹妹夺爱"
    scenes: List[Scene]

class Character:
    name: str               # "沈念安"
    role: str               # "主角" / "配角" / "龙套"
    appearances: List[str]  # 多套形象的图片URL
    voice_desc: str         # 音色描述

class SceneLocation:
    name: str               # "豪华酒店宴会厅"
    images: List[str]       # 多张场景参考图

class Storyboard:
    episode_id: str
    segment_index: int      # 片段序号
    shots: List[Shot]
    total_duration: float   # 总时长

class Shot:
    index: int
    duration: float         # 秒 (0-15s)
    time_of_day: str        # "日" / "夜"
    scene_ref: str          # @场景引用
    camera: str             # 镜头类型 (中景/近景/特写/过肩)
    camera_angle: str       # 拍摄角度描述
    characters: List[str]   # @角色引用列表
    action: str             # 角色动作描述
    dialogue: str           # 台词
    voice_style: str        # 音色描述
    background: str         # 背景描述
    camera_movement: str    # 镜头运动 (静止/推/拉/摇)
    transition: str         # 过渡方式
```
