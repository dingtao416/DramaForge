# DramaForge v2.0 — UI 界面设计规范文档

> 基于参考截图（小云雀平台）逐页解读，供前端代码生成使用。

---

## 一、全局设计语言

### 1.1 配色方案
- **主色**：紫色系 `#6C5CE7` / `#7C3AED`（用于高亮步骤、主按钮强调）
- **文字主色**：`#1A1A1A`（近黑色）
- **文字次色**：`#666666`
- **文字弱色**：`#999999` / `#BFBFBF`
- **背景色**：`#FAFAFA`（页面背景），`#FFFFFF`（卡片/面板）
- **边框色**：`#E8E8E8` / `#F0F0F0`
- **功能色**：成功绿 `#52C41A`，警告橙 `#FAAD14`，错误红 `#FF4D4F`

### 1.2 字体
- 中文：PingFang SC / Microsoft YaHei
- 英文：SF Pro / system-ui
- 等宽：JetBrains Mono

### 1.3 圆角体系
- 小按钮/标签：`4-6px`
- 卡片/输入框：`8-12px`
- 大卡片/弹窗：`12-16px`
- 圆形头像/按钮：`50%`

### 1.4 阴影
- 卡片悬浮：`0 2px 8px rgba(0,0,0,0.06)`
- 弹窗：`0 12px 24px rgba(0,0,0,0.12)`

### 3.2 左侧边栏详情
- **宽度**：240px，白色背景，右边框 1px
- **"+ 新对话"按钮**：满宽，高 40px，圆角 8px，浅灰背景+深色文字，左侧有 + 图标
- **"📁 资产库"链接**：图标+文字，字号 14px，灰色，hover 时背景变浅

- **项目条目**：32px 圆形头像（项目封面缩略图）+ 两行文字（标题+副标题），hover 时背景变浅
- **侧边栏右上角**：折叠按钮图标

### 3.3 主内容区 — 问候语
- **Logo**：48px 圆形图标，居中
- **问候语**：`Hi user..., 小云雀助你爆款马上成片`
  - 字号：28-32px，字重 600，居中
  - DramaForge 改为：`Hi, DramaForge 助你一键生成短剧`

### 3.4 主内容区 — 输入框
- **外框**：白色卡片，圆角 12px，带浅色阴影，hover/focus 时边框变紫色
- **文本域**：无边框，placeholder 灰色 `"告诉我，你今天想创造一点什么？"`
- **工具栏（底部）**：
  - 左侧：`[+]` 附件按钮 → `[🌐 Agent 模式 ▾]` 下拉 → `[📊 模型 ▾]` 下拉 → `[田]` 网格 → `[窗]` 窗口
  - 右侧：`[✨]` 设置图标 → `[⬆]` 发送按钮（圆形，灰色，输入内容后变黑/紫色）
- **发送按钮**：36px 圆形，无内容时灰色不可用，有内容时黑色可点击

### 3.5 快捷标签
- 输入框下方，居中排列
- 每个标签：圆角胶囊（border-radius 999px），浅灰边框，字号 13px
- 示例：`香港电影·996`、`猫狗偷玩手机一秒装睡`、`潮汕功夫茶宣传片`、`爆款手办生成`
- DramaForge 改为：`都市甜宠`、`古装权谋`、`悬疑探案`、`女频逆袭`、`男频热血`

### 3.6 常用功能卡片
- **标题**：`常用功能`，字号 16px，字重 600，左对齐
- **卡片横向排列**，可横向滚动，右侧有 `>` 箭头按钮
- 每张卡片：
  - 宽约 240px，高约 140px
  - **真实背景图片**（全覆盖，有暗色渐变蒙层）
  - 左下角文字：白色标题（16px 粗体）+ 白色副标题（12px）
  - 可选 `New` 绿色徽标在左上角
- DramaForge 四张卡片：
  1. `短剧 Agent` — Seedance 2.0 多剧集一键成片（New标签）
  2. `Seedance 2.0` — 首发试用
  3. `爆款复刻` — 自动解析爆点，参考文案/主题/画风
  4. `一镜到底` — 多张图片生成连续自然的转场
